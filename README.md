# EcoTrace 🌱

EcoTrace is a Carbon Footprint Tracking Platform developed during Virtual Prompt Wars by Hack2Skill under the Google Cloud Challenge.

## Features
- Carbon Footprint Calculator
- Interactive Dashboard
- Emissions Tracking
- Sustainability Action Plans
- Climate Education Resources

## Tech Stack
- React.js
- JavaScript
- HTML5
- CSS3
- Chart.js
